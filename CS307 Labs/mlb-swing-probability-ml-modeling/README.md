# MLB Swing Probability ML Modeling

## Overview

This folder contains a machine learning project focused on modeling swing probability in baseball data. It includes the working notebook, an exported report, a calibration script, and train/test parquet datasets.

## Key Files

- `mlb-swing-probability-notebook.ipynb` - main notebook for the modeling workflow.
- `mlb-swing-probability-ml-modeling-report.html` - exported report.
- `calibration.py` - supporting calibration-related Python script.
- `swing-train.parquet` - training dataset.
- `swing-test.parquet` - test dataset.

## Notes

- Start with the notebook for the end-to-end analysis.
- Use the HTML report for a quick read-through of the final write-up.
