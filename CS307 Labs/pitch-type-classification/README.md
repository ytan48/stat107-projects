# Pitch Type Classification

## Overview

This folder contains a baseball pitch type classification project. The work is centered around a notebook, an exported HTML report, and train/test parquet datasets.

## Key Files

- `pitch-type-classification-notebook.ipynb` - main notebook for the classification workflow.
- `pitch-type-classification.html` - exported report.
- `pitches-train.parquet` - training dataset.
- `pitches-test.parquet` - test dataset.

## Notes

- The notebook provides the primary implementation and analysis flow.
- The HTML file is useful when reviewing results without opening Jupyter.
