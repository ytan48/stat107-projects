# Credit Card Fraud Detection

## Overview

This folder contains materials for a credit card fraud detection lab project. The main analysis is stored in a Jupyter notebook and is supported by separate train and test parquet files.

## Key Files

- `credit-card-fraud-detection-notebook.ipynb` - primary notebook for the lab workflow.
- `fraud-train.parquet` - training dataset.
- `fraud-test.parquet` - test dataset.
- `credit-card-fraud-detection-report` - exported report.

## Notes

- The notebook is the best starting point for understanding the modeling steps used in this folder.
- The PDF provides the original assignment or lab context.
