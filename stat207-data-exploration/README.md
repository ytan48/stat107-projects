# STAT207 Data Exploration

## Overview

This folder contains STAT207 coursework centered on exploratory data analysis and modeling with video game data. Each subfolder holds a separate project stage, ranging from smaller assignments to the final project.

## Subfolders

- [Analysis of U.S. Video Games](analysis-of-us-videos-games/) - exploratory analysis work on the video game dataset.
- [Modeling Video Game Review Scores](modeling-video-game-review-scores/) - a follow-up modeling project using the same dataset.

## Notes

- All projects in this folder use Jupyter notebooks as the primary working format.
- The `video_games.csv` dataset is stored directly inside each project folder used by the notebook.
