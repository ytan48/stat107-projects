# Wine Quality ML Modeling

## Overview

This folder contains a machine learning project focused on wine quality modeling. It includes the main notebook, an exported report, and train/test parquet datasets.

## Key Files

- `wine-quality-ml-modeling-notebook.ipynb` - primary notebook for the analysis and model workflow.
- `wine-quality-ml-modeling-report.html` - exported report or notebook output.
- `wine-train.parquet` - training dataset.
- `wine-test.parquet` - test dataset.

## Notes

- The notebook is the main entry point for the project.
- The HTML report is useful for reviewing the analysis without running Jupyter.
