# STAT107 Data Science Discovery

## Overview

This folder contains coursework from STAT107: Data Science Discovery. It is split into smaller microprojects and larger course projects, with each subfolder documenting a specific topic or assignment.

## Subfolders

- [Microproject](Microproject) - a collection of short data science exercises focused on individual datasets or techniques.
- [Projects](Projects) - larger assignments with broader analysis or programming components.

## Notes

- Most work is stored in Jupyter notebooks.
- Some subprojects also include datasets, helper scripts, images, or requirement files.
- Open the subfolder README files for project-level summaries.
