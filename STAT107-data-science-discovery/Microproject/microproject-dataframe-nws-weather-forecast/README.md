# NWS Weather Forecast DataFrame Microproject

## Overview

This microproject contains a notebook that works with National Weather Service forecast data in a DataFrame-oriented workflow.

## Key Files

- `microproject-dataframe-nws-weather-forecast.ipynb` - main notebook.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The notebook contains both the data retrieval logic and the analysis or presentation steps.
