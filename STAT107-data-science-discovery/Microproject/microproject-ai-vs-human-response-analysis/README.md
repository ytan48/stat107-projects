# AI vs Human Response Analysis

## Overview

This microproject explores AI versus human response data using a notebook-based workflow. The folder includes the source CSV data, the notebook, and a local requirements file.

## Key Files

- `microproject-ai-vs-human-response-analysis.ipynb` - main notebook.
- `huggingface-hello-sampleAI-100.csv` - project dataset.
- `requirements.txt` - environment dependencies for the notebook.
