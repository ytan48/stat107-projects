# Boston Marathon Microproject

## Overview

This microproject analyzes Boston Marathon results across multiple years. The project is organized around a notebook and several CSV files containing race results.

## Key Files

- `microproject-boston-marathon.ipynb` - main analysis notebook.
- `results2000.csv` - marathon results dataset.
- `results2010.csv` - marathon results dataset.
- `results2023.csv` - marathon results dataset.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The CSV files provide multiple year snapshots that are used directly by the notebook.
