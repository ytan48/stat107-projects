# Choropleth Map DataFrame Microproject

## Overview

This microproject focuses on building or analyzing a choropleth map from a tabular dataset. The folder includes the notebook, the source CSV file, and dependency information.

## Key Files

- `microproject-choropleth-map-dataframe.ipynb` - main notebook.
- `uiuc-dmi-students-by-permanent-home-address.csv` - input dataset.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The notebook is the best place to review the mapping workflow and data preparation steps.
