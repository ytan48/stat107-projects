# World University Rankings Microproject

## Overview

This microproject analyzes world university ranking data using a notebook and a local CSV dataset.

## Key Files

- `microproject-world-university-rankings.ipynb` - main notebook.
- `World_University_Rank_2020.csv` - input dataset.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The CSV file is stored locally and is referenced directly by the notebook.
