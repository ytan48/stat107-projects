# Bechdel Test Microproject

## Overview

This microproject contains a notebook-based analysis related to the Bechdel Test. The folder keeps the notebook and dependency file together for a lightweight, self-contained assignment.

## Key Files

- `microproject-bechdel-test.ipynb` - main analysis notebook.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The notebook is the central artifact for both the data work and the written explanation.
