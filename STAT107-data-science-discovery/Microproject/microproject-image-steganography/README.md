# Image Steganography Microproject

## Overview

This microproject combines notebook work, Python code, and image assets for an image steganography exercise.

## Key Files

- `microproject-image-steganography.ipynb` - main notebook.
- `DISCOVERY.py` - supporting Python script.
- `discovery.png` - image asset used by the project.
- `i.png` - additional image asset.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The notebook and Python script work together in this folder more than in most of the other microprojects.
- Ignore the generated `__pycache__` directory when browsing the project.
