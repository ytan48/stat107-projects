# Roulette Martingale Microproject

## Overview

This microproject contains a simulation-based notebook exploring an infinite money in roulette or Martingale-style betting scenario.

## Key Files

- `microproject-infinite-money-in-roulette-martingale.ipynb` - main notebook.
- `requirements.txt` - environment dependencies for the notebook.

## Notes

- The notebook is the central artifact for both the simulation logic and the analysis.
